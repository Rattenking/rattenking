    步骤

    1. 确立一个学习目标；
    2. 理解要学习的知识；
    3. 以教代学，用输出代替输入；
    4. 进行回顾和反思；
    5. 实现知识的简化和吸收。

    确立一个学习目标

    1. 我们为什么要进行学习？

    在确定学习目标之前，我们首先需要明确学习的意义。

    使用费曼学习法的前提便是应该扭转对于学习的态度：学习不应是被动服从和由功利驱动的输入 ，而是我们自觉甚至是开心地实施有意识的主动学习，也就是以输出为载体的有选择的输入 。

    学习意义并不仅仅在于获得知识，其终极目标在于对于一个人思维的改变。

    费曼学习法对于思维的开放性、批判性、逻辑性以及更为清晰地表达观点这四个层面均能起到十分积极的作用。

    2. 如何确立并聚焦目标呢？

    1. 一方面，确立目标需要找到正确的方向。 对此，我们可以对自己提出的一些关键问题（比如未来的方向、当下的焦点），并且，要把对我们而言最重要的那件事作为自己的方向。

    2. 另一方面，确立目标需要找到真正的兴趣。 兴趣的发现和确定需要一个时间过程。

    3. 在确定目标之后，需要和目标建立强联系

    1. 第一，需要论证学习这门知识/做这件事的必要性。
    2. 第二，需要确认规划与目标的实质联系。在这个过程中，如何才能确保我们的目标是正确、而不是错误的呢？著名的 SMART 原则 提供了一个简单明了的判断标准。

    4. SMART 原则

    1. S：（Specific）目标必须清晰和可以形容
    2. M：（Measurable）目标必须量化和能够评估
    3. A：（Achievable）目标必须在能力范围内
    4. R：（Rewarding）目标必须有积极的意义
    5. T：（Time-bound）目标必须有实现的期限

    5. 目标遵循五个原则

    在指定具体目标的过程中，要让自己的目标遵循五个原则：

    1. 目标的全面性原则
    2. 目标的重点性原则
    3. 目标的挑战性原则
    4. 目标的可行性原则
    5. 目标的可调性原则

    理解要学习的知识

    在费曼学习法中，理解所要学习的知识需要遵循系统化原则——用系统化的思维去理解知识，归纳、筛选和分析知识，才能最终消化知识，为我所用。

    1. 归类和对比知识的来源

    1. 首先，系统化的第一步便是进行归纳，按照客观科学的逻辑，让不同来源的知识各归其位，以便于我们对比和筛选。

    2. 其次，筛选和留下最可靠的知识。 筛选知识包括三个流程：
      2.1 确立逻辑。明确学习的目的，并且建立客观科学的学习逻辑。
      2.2 收集信息。即锁定知识的来源、形成一个完整的知识框架。
      2.3 归类对比。筛选自己需要的知识以及保留可靠和重要的知识。

    最后，还有辨别假知识。具体可以先确立两到三个可靠的知识来源——老师、专业网站或图书馆，再把从这些来源得到的知识互相印证对比，找出对你重要的、不同领域的知识，再进行深度的学习。

    2. 形成一张思维和流程导图

    其中，思维导图成为对所学内容进行知识系统化的重要工具。思维导图的最大作用是可以让我们对知识进行横向的拓展，把不同的点以列表、图像、分支的形式展现在一张纸上，让知识的主要节点一目了然。

    人们在很大程度上低估了思维导图的作用。思维导图可以让知识可视化、知识场景的可视化、知识关系的可视化、学习过程的可视化。

    此外，思维导图提供的价值远不止于宏观角度对知识的理解，思维导图还深度参与了我们在学习时的认知加工的过程，可以帮助你在理解知识时更轻松与可控。

    3. 阅读与记忆的原则

    在阅读和记忆中，我们一边学习自己需要的知识，一边建设属于自己的思维框架，不但能加强对知识的理解，也会开始扩充自己的思维视野，同时在学习中积累对自己富有价值的知识点和技能点 。

    4. 第一次复述

    费曼学习法格外重视【输出】的作用——简单地讲，输出就是复述你所学到并理解到的知识并让听者理解。 第一次复述便是先讲给我们自己听一听。

    你要尝试着将学到的东西为自己讲解一遍，看能否像计划得那样理解，或者至少透彻地解读一大部分？

    具体来说，可以进行场景模拟，把自己当成是你的朋友，对着一面镜子，假设这是一个非常严肃的场合，对方想听一听你的看法，把书中精华的内容讲给他，确保他能完全听懂。

    我们学到的任何知识、接触的任何课程、视频乃至图片信息，你都可以用这种方式复述出来。

    这是对知识加深理解的最简单和最有效的方法，也是必不可少的步骤。通过第一遍复述，可以帮助我们建立建立长时记忆、加深对知识的理解、促进主动学习、对知识展开联想、得到关于问题的反馈。

    以教代学，用输出代替输入

    1. 为什么要进行以教代学？

    以教代学是费曼学习法的核心。在费曼学习法中，输入可以帮助输出，输出可以倒逼输入 。

    倒逼输入符合记忆学原理，最直接的好处是加强了对于特定内容的【留存率】 。

    以教代学要求以谁都能听懂的语言，进行简洁和有深度的分析。具体要求如下：

    1. 第一，语言简洁易懂。
    2. 第二，精准到位，没有歧义。
    3. 第三，讲出一定的深度。
    4. 第四，加上自己的理解。

    通过阐述知识的过程，反过来也可以强化自己对于知识尤其是重点内容的认知。

    2. 以教代学属于第二次复述

    在费曼学习法中，要格外重视第二次复述——第一次复述是把自己当作倾听者，第二次复述是进入一个真实的传授知识的场景，向别人甚至多个人阐述你对某项知识的看法。

    那么，如何进行第二次复述呢？

    开展复述时一定要贴近自己的兴趣，围绕自己的爱好：

    1. 我最感兴趣的部分是什么？（个人的目标）
    2. 我最擅长的讲述方式是什么？（个人的优势）
    3. 我最想跟对方交流的知识点是什么？（与外界的联系）

    围绕这三个答案去展开复述，在与别人的互动场景中探求知识，催化自己的深度学习。

    第二次复述的注意事项

    在第二次复述过程中，最好以分组讨论的方式进行，以便获得有效反馈和中肯评价。同时，第二次复述既需要阐述知识，也要展示自己的心灵品质，最好达到精神与知识达到了合二为一的状态。

    具体而言，在第二次复述过程中，要使用你自己的语言表述知识、结合现实阐述你对知识的解释、表达出你个人的分析和见解，而不是原封不动地背诵、复述。

    换而言之，第二次复述要为知识注入你个人的理解，并且通俗易懂地对别人讲出来。

    3. 建立稳定的输出系统

    为了养成以教代学的学习习惯，我们需要建立稳定的输出系统。输出过程是双向的，而不是单向的从一方向另一方传授知识。

    具体而言，在费曼技巧中，需要遵循五个输出原则：

    1. 对场景和思维模拟；
    2. 语言同于移动；
    3. 简洁的同时具有深度；
    4. 强化对重点知识点理解；
    5. 利用分组讨论获得反馈。

    这五个原则是一个递进同时又互相联系的学习过程，如果要实现学习的高效能，它们缺一不可。

    进行回顾和反思

    在费曼学习法中，回顾和反思起着承前启后的作用。这是从 学习知识 到 转化知识 的关键部分——如何将正确的知识从庞杂而具有欺骗性的信息中挑选出来。这种回顾和反思原则存在三个步骤：

    1. 第一步：怀疑和探索
    2. 第二步：寻找反证
    3. 第三步：加大内容留存率

    1. 第一步：怀疑和探索

    在复述完了之后回头检查、审视和总结。检查自己是否阐述正确，审视这些知识是否存在不易被发现的问题，并且总结前面几个环节的经验和教训。这需要我们重新对比数据和事实，即重新检查知识库、重新验证知识的关联。

    经过对比查证，会出现两种结果。

    1. 第一种结果是知识正确 ，我们再一次加强了对知识的理解。
    2. 第二种结果是知识不正确 。这时你就要谨慎地寻找原因。出现偏差的原因是什么？原知识中的哪些信息是存在问题的，为何第一遍学习时自己没有意识到这一点？通过寻找偏差和缺口，升华我们对于知识的理解。

    2. 第二步：寻找反证

    寻找反证的过程就是有目的地反思。反思不同于回顾和总结。反思则是对学习的质量进行解构，保证自己学到的是正确的知识。

    在反思过程中，需要做到以下四点：

    1. 一是重视否定式证据，比如相反的数据、逻辑漏洞、过时的知识、相反的权威观点。
    2. 二是当知识卡壳时，回到理解不清的地方，找出薄弱环节。
    3. 三是记住争议是深度学习的切入点和突破口。
    4. 四是没有最可靠的结论，要从科学、实用、系统的角度对知识进行评估。

    3. 第三步：加大内容留存率

    所谓内容留存率 ，这是指是我们记住、理解知识的比例。这个比例不低于90%才可称得上高效能的学习。

    对于学习而言，别一味地追求学习的数量，要着眼于学习的质量，提高学习的效能。

    实现知识的简化和吸收

    费曼技巧中最重要的一步就是简化并吸收所学的知识。

    简化知识的目的不是为了短时间内掌握某种技能，而是通过有效的简化，我们能全面而深入地理解自己的学习对象，将 有用的知识转化为自己的知识 。

    1. 如何简化知识？

    吾生也有涯，知无涯也。费曼向来不主张追求数量的贪多求全的学习。

    费曼认为，如果你不懂得简化所学的知识，就等于一直在盲目而缺乏方向地收集 碎片化知识 ，贪多不精，只积累了数量却做不到成体系地开发和利用。在学习过程中，好的东西过多，也会消化不良。

    简化知识存在两个要点：

    1. 一是打开知识的重要性开关。即明白哪些知识很重要，哪些知识一般重要，哪些知识不重要，为它们列一个优先级，排好顺序，全力吸收那些重要的知识。
    2. 二是将知识从复杂回归简单。即找到知识的核心逻辑，完善我们的思维模型，从知识中总结和提炼要点。

    具体操作过程中，存在两条简化原则：

    1. 一是对所学的知识进行系统性深挖；
    2. 二是形成自己的知识体系。在这两条简化原则之下，进行知识的吸收。

    2. 如何吸收知识？

    吸收知识分为两个方面：

    1. 一是，纵向拓展和精进知识；
    2. 二是深度挖掘，实现知识的内化。
    2.1 纵向拓展和精进知识

    理解新的知识不只是记下基本的原理，而是要将它们纳入现有的知识系统，要充分地掌握内在的规律。

    具体来说，纵向拓展和精进知识需要做到以下几点：

    1. 第一，纵向拓展；
    2. 第二，学习要有绿灯思维 ，即反对自我中心主义，以开放视野面对不同的学习观点；
    3. 第三，学习要以慢为快 ，尽量将知识融会贯通，而不是心浮气躁地囫囵吞枣；
    4. 第四，精进需要 刻意练习 。
    2.2 深度挖掘，实现知识的内化

    碎片化的知识片段对于我们成长无益。我们需要建立知识体系，将零碎分散、相对独立的知识和概念吸收转化为自己的东西，赋予它们逻辑，并且有效地运用这些知识。知识体系的建立的核心是知识的内化。

    对此，费曼提出五条具体建议：

    1. 第一，使用笔记记录知识的核心要素；
    2. 第二，一定要大幅度地整理所学的知识；
    3. 第三，对知识进行结构化的归纳与理解；
    4. 第四，输出和发布自己所理解的知识；
    5. 第五，对知识进行简化、吸收和记忆。

    3. 第三次复述

    费曼说： 我们所有形式的学习都是为了达到三个目的：

    1. 第一是解释问题，
    2. 第二是解决问题，
    3. 第三是预测问题。

    所谓第三次复述便是为了帮助读者在学习到一门知识后，可以同步达现这三个目标，能够使用自己所学的知识解释问题、解决问题和预测问题。

    人类的绝大部分知识都是通过学习获得。而第三次复述便是在学习的基础上，在已有的知识体系和思维训练的基础上，建立自己的原创观点，形成有影响力的新知识。

    这要求我们通过对于知识的创造能力检验我们的学习效果。这一步很难，却也是学习的最高层次。
