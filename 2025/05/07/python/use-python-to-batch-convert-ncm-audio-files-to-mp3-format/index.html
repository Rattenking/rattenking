    1. .ncm 格式解析

      • .ncm 是网易云音乐的加密音频格式，采用AES加密算法保护版权
      • 文件头包含关键信息：文件标识、版本号、密钥索引等

    2. ncmdump 工作原理

      • 解析文件头获取关键信息
      • 使用AES算法解密音频数据
      • 将解密后的数据转换为标准音频格式

    3. 性能优化建议

      • 使用多进程处理：利用 multiprocessing 模块并行处理多个文件
      • 内存优化：使用生成器处理大文件，减少内存占用
      • 错误重试机制：为失败任务添加重试逻辑

    4. 常见问题解决方案

      • 转换失败
        • 检查文件完整性，确保.ncm文件未损坏
        • 验证ncmdump版本是否最新
        • 检查系统时间是否正确，加密算法对时间敏感
      • 音质问题
        • 确保使用最新版FFmpeg
        • 调整音频比特率参数
        • 检查源文件是否为无损格式
      • 批量处理中断
        • 实现断点续传功能
        • 记录已处理文件状态
        • 添加异常捕获机制

    5. 安全注意事项

      • 不要在公共网络传输未加密的.ncm文件
      • 定期更新ncmdump和FFmpeg以修复安全漏洞
      • 谨慎处理转换后的音频文件，避免版权问题
